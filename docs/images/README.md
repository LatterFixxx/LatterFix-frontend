## Tips for Good Screenshots

1. **Use a consistent theme** - Dark or light mode, not mixed
2. **Use realistic data** - Redact sensitive information
3. **Highlight key features** - Use arrows or callouts if needed
4. **Optimize file size** - Compress PNGs for faster loading
5. **Use high resolution** - 2x for retina displays

## Creating Architecture Diagrams

Recommended tools:

- [Draw.io](https://app.diagrams.net/) - Free, web-based
- [Excalidraw](https://excalidraw.com/) - Hand-drawn style
- [Figma](https://figma.com) - Professional design tool
- [Mermaid](https://mermaid.js.org/) - Code-based diagrams
